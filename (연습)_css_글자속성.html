글자속성

글자에 관련된 속성은 글꼴, 글자크기, 글자 굵기, 기울기, 줄간격, 대소문자, 글자색상 등 주된 일곱가지를 필수적으로 알고 있어야한다.

color를 제외한 위에 6가지는 한줄쓰기 가능
속성명 종류 설명
font-family 컴퓨터에 등록되어있는 폰트명/ 웹폰트 활용가능 글꼴을 인용부호에 감싼다.
다국적 사용자를 대상으로 웹페이지를 제공할때
serif(명조체), snas-serif(고딕체), 글꼴 마지막에 작성
font font-size 기본 16px를 100% 기준으로 사용 수치에 px등의 단위를 붙여서 지정한다.
font-style italic /normal 글자의 기울기를 지정
font-variant small-caps /nomal 작은 대문자로 변경
font- weight bold /normal 호은 숫자 지정 400이 기본값/ 600이상은 볼드값 글자의 두께를 지정한다. font의 종류에 따라 달라지며 일정한 값은 아니다.
line-height 1은 100%의 값으로 줄간격 없음을 표기, 기본값은 1.4(140%) 글자의 높낮이를 따지는 행간을 조정한다.
font라는 속성으로 일렬쓰기 가능 (font-weight, font-style, font-variant) (font-size/line-height) (font-family)
세가지 순서대로 작성하돼, 글자크기와 글골은 생략할 수 없음
(괄호)안의 내용은 순서가 바뀔수 있고, 실제작성히 괄호는 쓰지 않음
font라는 속성이 우선순위이며 위에 이름값과 같지않더라도 덮어쓰기 가능
